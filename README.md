The coding challenge is done with React Typescript

## Author

Mohammed Mahmoud

#### Including :

- React.js
- Redux
- Jest

### Install node modules

```shell
yarn install
```

### Run development

```shell
yarn start
```

### To build production (you will find it in /build)

```shell
yarn build
```

### To run tests

#### Watch Test

```shell
yarn test
```
